Class ShowIfDrawer

Material property attribute that allows you to display the material property only if the specified material keywords are enabled.

Use with [ShowIf(KEYWORD_NAME, ...)] before a shader property. Most often used in combination with the [Toggle(KEYWORD_NAME)] attribute.

Examples

Use example:

    [ShowIf(_BASE_COLOR_ON)] _BaseColor("", Color) = (1,1,1,1)
